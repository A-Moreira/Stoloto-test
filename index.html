Необходимо, сверстать окно игры игра находится в модальном окне вкладки наверху переключаются

Бизнес логика:

Минимальное количество отмеченных чисел по четыре в каждом поле (кнопка «Подтвердить» пока не будет отмечено минимальное количество не активна)

При выборе минимальной ставки в поле Стоимость появляется цена 100 рублей, стоимость ставки отмеченные в каждой в вкладке суммируются, при выборе кол-ва тиражей, стоимость ставки перемножается на количество

При нажатии на кнопку «Очистить» очищаются все вкладки

При нажатии на кнопку «Случайные числа» в выбранной вкладке отмечаются минимальная ставка (т.е. по четыре в каждом вкладке)

При выполнении задания желательно использовать React/Redux, sass (можно scss), webpack, babel, (эти требования не обязательны можно использовать другие библиотеки, или нативный js)
